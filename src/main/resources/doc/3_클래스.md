## Kotlin 클래스

*  정의와 선언

        파일명과 클래스 명이 동일하지 않아도 됨
        한 파일내에 여러개의 클래스 생성 가능

* 가시성 변경자

        코틀린의 기본 가시성 변경자는 public
        접근자를 명시해주지 않으면 자동으로 public으로 선언됨

|변경자|클래스|최상위 함수,변수|
|------|---|---|
|public(기본)|모든 곳| 모든 곳|
|internal|같은 모듈 내| 같은 모듈 내|
|protected|하위 클래스 내|(사용불가)|
|private|같은 클래스 내| 같은 파일 내|
